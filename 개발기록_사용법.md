# 뜨개 도안 노트 – 개발기록 & 사용법

다른 컴퓨터에서도 이 파일을 수정·추가하면서 사용할 수 있습니다.

---

## 1. 프로젝트 개요

- **이름**: 뜨개 도안 노트 (Tuge)
- **형식**: HTML + CSS + JavaScript (빌드 없음, 브라우저에서 바로 실행)
- **저장소**: https://github.com/mereukk/Tuge  
- **배포 주소**: https://mereukk.github.io/Tuge/

---

## 2. 구현된 기능 정리

### 도안 작성
- **여러 도안 저장**: 사이드바에서 도안 이름으로 추가·선택·수정
- **도안 이름 변경**: 도안 옆 ⋮ 메뉴 → 이름 변경
- **도안 삭제**: ⋮ 메뉴 → 삭제 (숨겨져 있어 실수 방지)
- **1단, 2단…**: 단 추가, 각 단에 내용 입력
- **완료 체크**: 단 앞 체크박스 → 완료 시 딤드 + 취소선
- **구분선**: "구분선" 버튼으로 귀/머리/몸통/팔/다리 등 구간 구분, 구간마다 단수 1단부터 다시 시작
- **매직링**: 도안에 `mr` 로 입력 (상단 사슬/매직링 선택 UI는 제거됨)

### 코바늘 기호
- **기본 기호**: ch, s/s(빼뜨기), mr, sc, hdc, dc, tr, dtr, inc, dec, tog, sc2tog, dc2tog 등 (영문 약어)
- **기호 추가**: 코바늘 기호 페이지에서 이름·약어·설명 입력 후 추가, 도안 작성에서 바로 선택 가능
- **기호 팝업**: 도안 작성 중 "기호 참고 (팝업)" → **기호 넣기** 시 **약어만** 입력됨 (대괄호·한글 없음, 코 수에 반영)

### 도안 입력 보조
- **색 구분**: 숫자(빨강), 영문 약어(초록), 괄호(보라)로 표시
- **한글 해석**: ch1 → 사슬 1개, sc4 → 짧은뜨기 4코 등으로 해석 표시
- **총 코 수**: 각 단별로 자동 계산
  - 대괄호 `[]` 안 내용은 제외
  - **빼뜨기(s/s)**, **매직링(mr)** 은 코 수에 포함하지 않음
- **괄호 반복**: `(sc4 sc2tog1)*6` 형태 계산 지원

### 저장·불러오기
- **저장 방식**: **로컬(브라우저) 저장** — 서버/사이트에 올라가지 않음. 같은 기기·같은 브라우저에서만 유지됨.
- **도안 전체 저장**: 사이드바 좌측 하단 "전체 저장" → JSON 파일로 다운로드 (백업용)
- **불러오기**: "불러오기" → 저장한 JSON 선택 시 도안 목록 교체
- 다른 기기에서 쓰려면 전체 저장 → 파일 보관 → 그 기기에서 불러오기.

### UI
- **사이드바**: 전체 고정(스크롤해도 안 움직임). **스크롤 독립**: 본문만 스크롤, 사이드바 내부는 도안 목록만 스크롤. **전체 저장·불러오기**는 좌측 하단 고정.
- **사이드바**: 숨기기(◀), 다시 열기(도안 ▶ 탭), 좌우 드래그로 너비 조절
- **복사**: "도안 복사"로 현재 도안 텍스트 클립보드 복사

## 3. 로컬에서 사용하기

1. `index.html` 을 브라우저로 연다.
2. 도안 작성 / 코바늘 기호 메뉴로 이동해 사용한다.
3. 데이터는 해당 브라우저의 localStorage에만 저장된다 (다른 기기와 자동 동기화 안 됨).

## 4. GitHub에 푸시하기 (이 컴퓨터)

### 처음 한 번만
```cmd
git config --global user.email "leehaae11@gmail.com"
git config --global user.name "mereukk"
```

### 푸시 (권장)
- **`run-push.bat`** 더블클릭 권장. 창이 자동으로 안 닫혀서 로그·에러를 확인하기 좋음.
- 또는 **`push-to-github.bat`** 더블클릭 (실행 후 창이 바로 꺼지면 아래 "푸시 배치 컨텍스트" 참고).
- 터미널에서 실행할 때:
  ```cmd
  cd "프로젝트 폴더 경로"
  push-to-github.bat
  ```
- 실패 시 창에 나온 에러 메시지 확인. 성공 시 "Done." 후 창 유지(또는 run-push.bat 사용 시 직접 창 닫기).

### 푸시 배치 컨텍스트 (문제 해결용)
- **run-push.bat**: `cmd /k`로 push-to-github.bat을 실행해, 끝나도 창이 닫히지 않음. **더블클릭은 이 파일 사용 권장.**
- **push-to-github.bat**: Git 확인 → add/commit → remote 추가(없을 때) → fetch → **rebase origin/main** → push.  
  - 한글을 넣으면 CMD 인코딩 때문에 명령이 깨져 창이 바로 꺼질 수 있음 → **메시지는 영문만 사용.**
  - **머지 충돌 표시**(`<<<<<<< HEAD`, `=======`, `>>>>>>>`)가 파일에 들어가 있으면 CMD가 잘못 해석해 창이 바로 꺼짐.  
    pull/merge 후 배치가 깨졌다면 이 문서나 저장소에서 깨끗한 버전으로 덮어쓰기.
  - push가 **non-fast-forward**로 거절되면, 원격 변경을 받은 뒤 rebase로 로컬 커밋을 그 위에 올리도록 배치에 이미 반영해 둠 (fetch → rebase origin/main → 실패 시 merge with --allow-unrelated-histories).
- **바로 꺼질 때**: `run-push.bat` 사용. 그래도 꺼지면 push-to-github.bat에 충돌 문구가 있는지 확인 후, 있으면 제거하거나 이 문서 기준으로 배치 내용 복구.

## 5. GitHub Pages 배포

- **저장소**: https://github.com/mereukk/Tuge  
- **설정**: Settings → Pages → Source: Deploy from a branch → Branch: **main** → / (root) → Save  
- **주소**: https://mereukk.github.io/Tuge/  
- 배포 반영까지 1~2분 걸릴 수 있음. 수정이 안 보이면 **Ctrl+F5** 강력 새로고침(캐시 무시).

## 6. 다른 컴퓨터에서 수정하기

1. **저장소 받기**
   - https://github.com/mereukk/Tuge 접속 → Code → Download ZIP  
   - 또는 Git 설치된 PC: `git clone https://github.com/mereukk/Tuge.git`
2. **수정**
   - 받은 폴더에서 `index.html`, `app.js`, `style.css` 또는 이 `개발기록_사용법.md` 등 원하는 파일 수정.
3. **다시 GitHub에 올리기**
   - 그 컴퓨터에서도 한 번만:
     ```cmd
     git config --global user.email "leehaae11@gmail.com"
     git config --global user.name "mereukk"
     ```
   - 해당 폴더에서:
     ```cmd
     git add .
     git commit -m "다른 PC에서 수정"
     git push origin main
     ```
   - 또는 그 폴더에 `push-to-github.bat` 을 복사해 두고 더블클릭해도 됨 (원격이 이미 origin으로 잡혀 있으면 push만 실행).

이렇게 하면 다른 컴퓨터에서도 이 파일과 프로젝트를 수정하고 GitHub에 푸시할 수 있습니다.

## 7. 파일 구성

| 파일 | 설명 |
|------|------|
| index.html | 메인 페이지 (메뉴, 도안 작성, 기호 페이지, 팝업) |
| style.css | 스타일 |
| app.js | 도안·기호·계산·저장 등 전부 로직 |
| README.md | 프로젝트 짧은 소개 |
| 개발기록_사용법.md | 이 문서 (기능 정리, 사용법, 푸시, 다른 PC 수정) |
| push-to-github.bat | Windows에서 GitHub 푸시용 배치 (fetch → rebase → push) |
| run-push.bat | 푸시 실행기. 더블클릭 시 창이 안 꺼지게 push-to-github.bat 실행 (권장) |
| push-to-github.ps1 | PowerShell 푸시 스크립트 (실행 정책 제한 시 .bat 사용) |
| .gitignore | Git 제외 파일 |
