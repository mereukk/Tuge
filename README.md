# 뜨개 도안 노트

코바늘 기호를 정리하고 도안(1단, 2단…)을 적을 수 있는 간단한 웹 페이지입니다.

## 사용 방법

1. `index.html` 파일을 브라우저에서 열어서 사용하세요.
2. **도안 작성**: 시작을 사슬/매직링 중에서 고르고, 단을 추가해서 각 단 내용을 입력합니다.
3. **코바늘 기호**: 메뉴에서 "코바늘 기호"를 누르면 기본 기호 목록과 기호 추가 폼이 나옵니다. 여기서 추가한 기호는 도안 작성에서도 선택할 수 있습니다.
4. **기호 참고 (팝업)**: 도안 작성 화면에서 "기호 참고 (팝업)"을 누르면 기호 목록이 팝업으로 열립니다. 입력 칸에 포커스한 뒤 팝업에서 "선택"을 누르면 해당 기호가 도안에 삽입됩니다.
5. **도안 복사**: 작성한 도안을 텍스트로 복사할 수 있습니다.

데이터(기호 목록, 도안)는 브라우저 로컬 저장소에 저장됩니다.
