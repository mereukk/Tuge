# 뜨개 도안 노트 (Tuge)

코바늘 기호를 정리하고 도안(1단, 2단…, 구간 구분)을 적을 수 있는 웹 앱입니다.

- **실행**: `index.html` 을 브라우저로 열기
- **배포**: https://mereukk.github.io/Tuge/
- **푸시**: `run-push.bat` 더블클릭 (창 유지). 상세: [개발기록_사용법.md](./개발기록_사용법.md)

데이터(기호, 도안)는 브라우저 localStorage에 저장되며, "전체 저장"으로 JSON 백업 가능합니다.

**코 수 규칙**: 빼뜨기(s/s)·매직링(mr)은 코 수 제외. `inc` 앞에 붙여쓰기(scinc, hdcinc, dcinc 등)는 1개당 2코. 자세한 기능·문제 해결: [개발기록_사용법.md](./개발기록_사용법.md)
